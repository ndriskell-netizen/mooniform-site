Mooniform — Live Tech

Advance info for small–medium stages. Adaptable as needed.

Band: Mooniform
From: Abilene, TX
On stage: 6 people
Set length: 45–90 min (no breaks)

Inputs (typical)

  • Vocals: up to 5
    • usually 2–4 active
    • two can share a mic if needed
  • Guitars: 2 (amp’d, mono)
  • Bass: amp mic preferred (DI OK)
  • Synth: 1 mono out
  • Drums: kick / snare / 3 toms / OHs
  • Electronic drum pad: 1 input (intermittent, optional)

No backing tracks. No playback rig.

Monitoring

  • Drums: in-ears
    • needs one dedicated monitor mix via XLR
    • includes click (generated on stage)
  • Everyone else: wedges are fine

Stage / Practical

  • Fixed positions
  • Small–medium stages OK
  • Prefer 15–20 min minimum for changeover
  • Strongly prefer not to backline drums
  • Open to simplifying inputs (esp. vocal mics) if needed

Optional

  • Drummer can bring a full Earthworks drum mic kit
    (or use the house drum mics if preferred)
